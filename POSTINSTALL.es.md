# Content Studio está listo

Organiza tu calendario de contenido desde una vista mensual simple: crea posts,
asígnales fecha, canal, categoría, color, caption y estructura para imagen o
video.

Puedes empezar por:

- Crear categorías para tus pilares, campañas o formatos.
- Agregar posts directamente desde el calendario.
- Arrastrar posts entre días para ajustar tu planificación.
- Adjuntar archivos o referencias a cada post.
- Usar el Planificador para pedirle al Agent que proponga semanas de contenido,
  ajuste tonos o transforme ideas en posts.

Todo queda guardado localmente en el espacio privado de la app.
